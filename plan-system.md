# Hawk Eye - نظام التحليل الجنائي الذكي

## المشروع
بناء نظام ويب متكامل (React SPA) للتحليل الجنائي لبيانات CDR مع:
- لوحة تحكم متكاملة مع واجهة Prompt نصية
- محاكاة 300 وكيل تحقيق (300 Agent System)
- تحليل متقدم للمقاطعات والحوار
- تصور بياني تفاعلي

## المتطلبات
- Frontend: React + TypeScript + Tailwind + shadcn/ui + Vite
- Charts: Recharts + D3.js
- Animation: Framer Motion
- State: React Context + useReducer

## الصفحات
1. **Home/Landing** - الصفحة الرئيسية
2. **Dashboard** - لوحة التحكم الرئيسية مع Upload وتحليل
3. **Agent Swarm** - صفحة 300 وكيل (محاكاة بصرية + تحليل)
4. **CDR Analysis** - تحليل تفصيلي لبيانات CDR
5. **Cross-Case** - تحليل المقاطعات بين القضايا
6. **Reports** - التقارير والنتائج
7. **Prompt Lab** - واجهة الأوامر النصية الذكية

## الـ 300 وكيل
- 100 وكيل تحليل بيانات (Data Analysts)
- 100 وكيل تحري وبحث (Investigators)
- 100 وكيل بناء حوار واستنتاج (Dialogue Builders)
