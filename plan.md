# خطة تحليل قضية السطو المسلح - دار أحمد / الشامية حي المعلمين

## معلومات الحادث
- **نوع الجريمة**: سطو مسلح
- **الوقت**: 3:30 - 4:20 (50 دقيقة)
- **الموقع**: دار أحمد، الشامية - حي المعلمين
- **الهدف**: تحليل بيانات الاتصالات والأبراج لتحديد المشتبه بهم

## المراحل

### المرحلة 1: قراءة ودمج الملفات
- قراءة 11 ملف Excel المرفوعة
- فهم هيكل كل ملف (الأعمدة، أنواع البيانات)
- دمج الملفات في قاعدة بيانات موحدة

### المرحلة 2: فهم البيانات والتنظيف
- تحديد أنواع البيانات: CDRs، مواقع الأبراج، إيميلات، مكالمات
- تنظيف البيانات وتوحيد التنسيقات
- تحديد المواقع الجغرافية لكل ملف/منطقة

### المرحلة 3: التحليل الجنائي
- تحديد الأبراج القريبة من موقع الحادث (حي المعلمين - دار أحمد)
- تصفية البيانات حسب فترة الحادث (3:30 - 4:20)
- استخراج الأرقام التي تواجدت في موقع الحادث
- تحليل خطوط السير (Movement Patterns)

### المرحلة 4: التقاطعات والارتباطات
- إيجاد الأرقام المتقاطعة بين الأبراج المختلفة
- تحليل المكالمات والرسائل خلال فترة الحادث
- استخراج الإيميلات المرتبطة
- إيجاد الأرقام "الغريبة" غير المرتبطة بالمنطقة

### المرحلة 5: التقرير النهائي
- إعداد تقرير تحليلي شامل
- جداول الأرقام المشتبه بهم
- خريطة خطوط السير
- الإيميلات والمكالمات المرتبطة
